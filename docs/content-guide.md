# コンテンツガイドライン

## 掲載基準

以下の条件を満たすものを語録として掲載する。

- 涼花みなせさん本人が配信・動画で発言したフレーズであること
- 複数の配信で使われているか、特に印象的な場面で使われたものであること
- 配信外（プライベートな発言など）のものは対象外

判断に迷う場合はモデレーター間で相談する。

## ファイルの作成方法

`src/content/quotes/` フォルダにMarkdownファイルを作成する。

**ファイル名のルール:**
- ローマ字表記を使う（例: `mushikera.md`、`otsumina.md`）
- スペースは使わず、単語をつなげるかハイフンで区切る
- すでに同じ語録のファイルがないか確認してから作成する

## Markdownのテンプレート

```markdown
---
title: 語録（日本語）
reading: よみがな
meaning: 一言で意味を説明
tags: [タグ1, タグ2]
rarity: 3
first_date: ""
first_appearance: ""
youtube_url: ""
image: ""
contributor: []
usage:
  - text: ""
    date: ""
youtube_urls:
  - label: ""
    url: ""
---

## 解説

語録が生まれた背景・経緯・エピソードを書く。
```

## 各フィールドの説明

| フィールド | 必須 | 説明 |
|---|---|---|
| `title` | ○ | 語録そのもの |
| `reading` | ○ | ひらがなで読み方 |
| `meaning` | ○ | 短く分かりやすく意味を説明 |
| `tags` | ○ | 後述のタグ一覧から選ぶ |
| `rarity` | ○ | 1〜5の数字（後述） |
| `first_date` | △ | 初出日。`YYYY-MM-DD` 形式。年・月のみも可。不確かな場合は末尾に `?` を付ける（例: `2024-03?`）。空欄可 |
| `first_appearance` | △ | 分かる範囲で配信タイトル・日付を記載。空欄可 |
| `youtube_url` | △ | 初出のYouTube URL。空欄可 |
| `image` | △ | 配信シーンのスクリーンショット。`/images/quotes/ファイル名.jpg` の形式で記載。空欄可 |
| `contributor` | △ | 投稿・情報提供者のハンドルネーム。複数人可 |
| `usage` | △ | 実際の使用例。`text`（セリフ）と`date`（日付、任意）をセットで記載。複数可。空欄可 |
| `youtube_urls` | △ | 関連動画のリンク（初出以外）。`label` と `url` をセットで記載。空欄可 |

空欄の場合は `""` または `[]` のまま残してもエラーにはなりません。

## レア度（rarity）の基準

| 値 | 目安 |
|---|---|
| 1 ★☆☆☆☆ | 新規ファン必見 |
| 2 ★★☆☆☆ | よく使う |
| 3 ★★★☆☆ | 印象的・記憶に残る |
| 4 ★★★★☆ | 知る人ぞ知る |
| 5 ★★★★★ | 激レア・名言・格言 |

主観でよい。迷ったら2か3に設定しておき、後から変更する。

## タグの一覧

自由に付けてよい。以下は現在使われているタグの例。

| タグ | 説明 |
|---|---|
| 挨拶 | 配信の開始・終了時の挨拶 |
| 感嘆詞 | 驚きや喜びを表す言葉 |
| 罵倒 | （愛情を込めた）悪口・ツッコミ |
| ゲーム | ゲーム配信中に生まれたフレーズ |
| コラボ | 他の配信者とのコラボで生まれたフレーズ |
| オリジナル曲 | オリジナル楽曲に関連するフレーズ |
| 生き様 | みなせさんの考え方・人生観を表す言葉 |
