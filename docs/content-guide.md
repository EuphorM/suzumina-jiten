# コンテンツガイドライン

## 掲載基準

以下の条件を満たすものを語録として掲載する。

- 涼花みなせさん本人が配信・動画で発言したフレーズであること
- 複数の配信で使われているか、特に印象的な場面で使われたものであること
- 配信外（プライベートな発言など）のものは対象外

判断に迷う場合はモデレーター間で相談する。

## ファイルの作成方法

`src/content/quotes/` フォルダにMarkdownファイルを作成する。

**ファイル名のルール:**
- ローマ字表記を使う（例: `mushikera.md`、`otumina.md`）
- スペースは使わず、単語をつなげるかハイフンで区切る
- すでに同じ語録のファイルがないか確認してから作成する

## Markdownのテンプレート

```markdown
---
title: 語録（日本語）
reading: よみがな
meaning: 一言で意味を説明
tags: [タグ1, タグ2]
rarity: 3
first_appearance: "配信タイトルや日付"
youtube_url: "https://www.youtube.com/..."
---

## 解説

語録が生まれた背景・経緯・エピソードを書く。

## 使用例

- 実際に使われた文脈や台詞
- 複数ある場合は箇条書きで追加
```

## 各フィールドの説明

| フィールド | 必須 | 説明 |
|---|---|---|
| `title` | ○ | 語録そのもの |
| `reading` | ○ | ひらがなで読み方 |
| `meaning` | ○ | 短く分かりやすく意味を説明 |
| `tags` | ○ | 後述のタグ一覧から選ぶ |
| `rarity` | ○ | 1〜5の数字（後述） |
| `first_appearance` | △ | 分かる範囲で配信タイトル・日付を記載 |
| `youtube_url` | △ | YouTube のURLがあれば記載 |

## レア度（rarity）の基準

| 値 | 目安 |
|---|---|
| 1 ★☆☆☆☆ | 定番・頻繁に使う |
| 2 ★★☆☆☆ | たまに使う |
| 3 ★★★☆☆ | 印象的な場面で登場した |
| 4 ★★★★☆ | なかなか聞けない |
| 5 ★★★★★ | 激レア・一度しか聞いたことがない |

主観でよい。迷ったら低めに設定しておき、後から変更する。

## タグの一覧

現在使用しているタグ。新しいタグが必要な場合はモデレーター間で相談して追加する。

| タグ | 説明 |
|---|---|
| 挨拶 | 配信の開始・終了時の挨拶 |
