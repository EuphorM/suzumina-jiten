# 運営に参加するには

すずみな語録辞典はファン有志で運営しています。一緒に育ててくれる方を歓迎します。

## 参加できること

| 役割 | 内容 | エンジニア不要 |
|---|---|---|
| 語録のレビュー | 投稿された語録の内容を確認・掲載判断する | ✅ |
| 語録の追加・編集 | GitHubのWeb UIからMarkdownファイルを作成・修正する | ✅ |
| サイトの改善 | デザインや機能の修正・提案 | 提案のみなら✅ |

## 参加方法

1. [GitHub](https://github.com/) のアカウントを作成する（すでにお持ちの方はそのままどうぞ）
2. [Issues ページ](https://github.com/EuphorM/suzumina-jiten/issues) を開く
3. 右上の「New issue」ボタンをクリック
4. 「運営への参加希望」をクリック
5. 「Create」をクリック

内容を確認のうえ、GitHubのコラボレーターとして招待します。

## 参加後の流れ

招待を承認すると、語録の追加・編集・Issueの管理ができるようになります。

詳しい操作方法は [モデレーター運用マニュアル](./moderator-guide.md) をご覧ください。
