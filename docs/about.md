# すずみな語録辞典について

**サイト名:** すずみな語録辞典（非公式）

**目的:** 涼花みなせさんの配信で生まれた語録・印象的なフレーズを体系的に記録する

## なぜ作ったか

涼花みなせさんは独特なフレーズを使うことがある。既存のファンには「また言ってるｗ」と笑えるものでも、新しい視聴者には「どういう意味？」と疑問に思われることがありました。その疑問に答えられる場所を作りたい——それがこのサイトを作り始めたきっかけです。

## 誰のためのサイトか

新しく配信を見始めた人には、語録から意味や由来、その元配信にまでたどれるように。既存のファンには、語録を通じてみなせさんとの思い出を振り返ることができるように。

## みなせさんとファンの文化を残す

涼花みなせさんはファンをとても大切にしていて、ファンもまたみなせさんをとても大切にしています。

その空気感が配信をやさしい場所にしていて、すべてが許されるからこそ、みなせさんらしさが溢れた独特なフレーズになっていることもあります。

語録にはそういう瞬間が詰まっています。言葉だけでなく、その空気ごと残していけるように。

## みんなで育てる辞典

投稿フォームには「うろ覚えでも大丈夫です！」と書きました。

完璧な情報でなくてもいいので、ファンが少しずつ持ち寄って、みんなで育てていく辞典にしたいです。

みなせさんの語録には、日々の配信の記憶が残っています。だからこそ、ファンの間で愛されています。

## 運営の方針

運営はファン有志で行っています。語録をご存知の方は投稿フォームからお気軽にお知らせください。投稿いただいた内容はレビューのうえ掲載しています。

投稿内容のレビュー・掲載判断、サイトの改善など、運営側として参加してくれる方も大募集しています！エンジニアでなくても運営できるよう設計していますので、興味のある方は[こちら](https://github.com/EuphorM/suzumina-jiten/blob/main/docs/contributing.md)をご覧ください。

サイトの制作内容も著作権フリーで公開しています。ぜひご自由にお使いください。

## 非公式であることについて

このサイトは涼花みなせさんの非公式ファンサイトです。涼花みなせさん本人とは関係なく、ファンが楽しんで作っています。

掲載している語録の権利は涼花みなせさんに帰属します。このサイトはあくまでファンによる記録であり、みなせさんへのリスペクトを前提として運営していきます。
