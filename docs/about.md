# すずみな語録辞典について

**サイト名:** すずみな語録辞典（非公式）

**目的:** 涼花みなせさんの配信で生まれた語録・印象的なフレーズを体系的に記録する

## なぜ作ったか

涼花みなせさんは配信中、独特なフレーズを使うことがあります。既存のファンには笑えるものでも、新しい視聴者には「どういう意味？」と疑問に思われることも。その疑問に答えられる場所を作りたい——それがこのサイトを作り始めたきっかけです。

初めての人には、語録の意味や由来、その元配信にまでたどれるように。長く一緒にいるファンには、語録を通じて思い出を振り返れる場所になるように。

## みなせさんとファンの文化を残す

みなせさんはファンをとても大切にしていて、ファンもまたみなせさんのことをとても大切に思っています。

その空気が配信をやさしい場所にしていて、すべてが許されるからこそ、みなせさんのありのままが語録になっています。

## みんなで育てる辞典

完璧な情報じゃなくても、みんながちょっとずつ持ち寄ってくれたら嬉しいです。

共に過ごした記憶も詰まっているから、多くのリスナーに愛されています。

## 運営の方針

運営はファンの有志で行っています。語録をご存知の方は投稿フォームからお気軽にお知らせください。投稿いただいた内容はレビューのうえ掲載しています。

投稿内容のレビュー・掲載判断、サイトの改善など、運営側として参加してくれる方も大募集しています！エンジニアでなくても運営できるよう設計していますので、興味のある方は[こちら](https://github.com/EuphorM/suzumina-jiten/blob/main/docs/contributing.md)をご覧ください。

このサイトは非公式ファンサイトです。掲載している語録の権利は涼花みなせさんに帰属します。みなせさんへのリスペクトを前提として運営していきます。サイトのデザイン・コード等は全て著作権フリーです。
