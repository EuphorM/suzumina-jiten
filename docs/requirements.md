# 要件定義書

## プロジェクト概要

**サイト名:** すずみな語録辞典（非公式）  
**目的:** 涼花みなせさんの配信で生まれた語録・印象的なフレーズを体系的に記録・公開する

## 背景

涼花みなせさんは配信において独特なフレーズを多く用いる。既存の視聴者にとっては親しみ深いが、新規の視聴者には意味が分からず「どういう意味ですか？」という質問が配信コメントに出ることがあった。このサイトはその課題を解決するとともに、サイトを経由して涼花みなせさんを楽しんでもらう入口にすることを目指す。

## 対象ユーザー

| ユーザー | 目的 |
|---|---|
| 新規視聴者 | 語録の意味・背景を調べる |
| 既存視聴者 | 語録を投稿・共有して楽しむ |
| モデレーター | 投稿された語録を審査・掲載する |

## 機能要件

### 閲覧機能
- 語録の一覧表示（タイトル・読み・意味・タグ・レア度）
- 語録の詳細表示（解説・使用例・YouTube配信リンク）

### 投稿機能
- サイト内フォームから語録を投稿できる
- 投稿内容は GitHub Issue として自動記録される

### 管理機能
- モデレーターが投稿内容を確認・承認・却下できる
- 承認された語録はMarkdownファイルとして追加・公開される

## 非機能要件

- モデレーターはエンジニアでなくても操作できること
- コンテンツはテキストファイル（Markdown）で管理し、誰でも内容を確認できること
- サイトの表示は高速であること（静的サイト）
- 運用コストは無料または最小限に抑えること
